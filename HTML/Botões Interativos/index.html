10 Botões Interativos

HTML

No HTML, criamos a estrutura básica do site com uma <div> principal contendo uma seção de informações e uma seção de botões. Cada botão é criado com a tag <button> e atribuímos a cada um uma classe específica para aplicar estilos diferenciados. As funções buttonFunctionX() são associadas ao evento de clique dos botões e podem ser definidas no arquivo script.js.

CSS

No CSS, aplicamos estilos para os botões e o layout geral da página. Aqui estão os principais aspectos:

  • * { margin: 0; padding: 0; box-sizing: border-box; }: Remove margens e preenchimento padrão e define o box-sizing como border-box para facilitar o controle do tamanho dos elementos.
  • body { font-family: Arial, sans-serif; background-color: #ffffff; }: Define a fonte padrão e a cor de fundo da página como branca.
  • .container: Centraliza o conteúdo da página, aplica uma cor de fundo leve, bordas arredondadas e uma sombra para criar um efeito de profundidade.
  • .title-section: Alinha o título no centro e adiciona uma margem inferior para espaçamento.
  • .button-container: Organiza os botões em uma grade flexível, garantindo que eles se ajustem ao tamanho da tela e estejam centralizados.
  • .button: Define o estilo básico dos botões, incluindo tamanho, cor do texto, bordas, arredondamento, tamanho da fonte e efeitos de transição para interações suaves.
  • background-color: Define a cor de fundo dos botões. Cada botão possui uma cor distinta para facilitar a identificação.
  • border-radius: Arredonda os cantos dos botões para um visual mais suave.
  • transform: Utilizado para aplicar efeitos de animação quando o mouse passa sobre o botão, como scale (ampliar), rotate (rotacionar) e translate (mover).
  • transition: Adiciona animações suaves para as transformações e mudanças de cor, melhorando a experiência do usuário.
  • box-shadow: Cria uma sombra ao redor dos botões durante o efeito de hover, destacando ainda mais a interação.

Esses estilos combinados ajudam a criar uma interface interativa e visualmente atraente.

JavaScript

No arquivo script.js, definimos funções JavaScript que são chamadas quando os botões são clicados. Essas funções podem ser personalizadas para executar ações específicas, como exibir mensagens ou manipular o DOM.
